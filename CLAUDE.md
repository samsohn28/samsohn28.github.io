# Site

Jekyll site hosted on GitHub Pages. No build step needed locally.

# Blog posts

After writing or editing any post in `blog/`, run the humanizer skill on it before committing.

# Style

- Match existing post voice: short sentences, first person, self-deprecating, no AI-isms
- Section headings: short, lowercase (e.g. "The data", "Where it stands")
- Never add comments, docstrings, or unnecessary files
